# EMG Projektbericht

Dieses Projekt enthält den aktuellen Projektbericht als LaTeX-Dokument.

Struktur:
- `main.tex`: Hauptdatei
- `pages/`: Frontmatter-Seiten wie Deckblatt, Abstract und Erklärung
- `chapters/`: Kapitel des Berichts
- `figures/`: Grafiken und Zeichnungen
- `bibliography.bib`: Literaturdatenbank

Build:
- `make pdf` baut die PDF-Datei in `build/`
- `make clean` entfernt temporäre Dateien
- `make purge` löscht den gesamten `build/`-Ordner
